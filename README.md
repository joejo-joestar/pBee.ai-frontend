## Using this repo

creating react project
```terminal
npm create vite@latest

npm install
npm cd [project name]
```

installing other dependencies
```terminal
<!-- React-Router -->
npm i react-router-dom

<!-- Framer -->
npm i framer-motion

<!-- Other Dependencies -->
npm i -D @types/node

<!-- Tailwind Install -->
npm i -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Test Server
```terminal
npm run dev
```
