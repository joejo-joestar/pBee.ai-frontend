## Using this repo

creating react project
```terminal
npm create vite@latest

npm install
npm cd [project name]
```

installing other dependencies
```terminal
npm i react-router-dom

npm i framer-motion
npm i -D @types/node

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Test Server
```terminal
npm run dev
```
